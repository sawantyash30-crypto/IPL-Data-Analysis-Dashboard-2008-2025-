# 🏏 IPL Data Analysis Dashboard (2008–2025)

## 📌 Project Overview

This project focuses on analyzing Indian Premier League (IPL) data from 2008 to 2025 to uncover key insights about team performance, player statistics, and overall match trends.

An interactive dashboard was created to present the data in a clear and visually appealing way, enabling better understanding and decision-making.

---

## 📊 Key Insights

* 🏆 2025 Champion: Royal Challengers Bangalore
* 🥈 Runner-Up: Punjab Kings
* 🔥 Orange Cap: B Sai Sudharsan (759 Runs)

* 💥 Total 6’s: 1296
* ⚡ Total 4’s: 2251
